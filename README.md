# PowerBI-Sales-Business-Insights

## 📌 Project Overview
This repository contains a comprehensive, multi-page Business Intelligence solution developed as part of an Agile team of 5 during my Data Analyst Internship at Capgemini.

The project focuses on transforming raw business data into meaningful insights across multiple dimensions including **executive performance, product analysis, regional trends, customer behavior, operations, and profitability to support data-driven decision-making.

---

## 📊 Dashboards Overview

This Power BI project consists of 5 interactive dashboards:

### 1. Executive / Overview Dashboard
Provides a high-level summary of business performance using key KPIs such as Revenue, Profit, and Units Sold for quick executive decision-making.

### 2. Product Analysis Dashboard
Analyzes product-level performance to identify top-performing and underperforming products across categories.

### 3. Region-wise Analysis Dashboard
Displays geographic performance insights to understand revenue and sales distribution across different regions.

### 4. Customer & Operational Dashboard
Focuses on customer insights and operational efficiency metrics to evaluate business processes and user behavior.

### 5. Profit Dashboard
Provides detailed profitability analysis including margins, cost breakdowns, and key financial drivers affecting business performance.

## ⚙️ Technologies Used

- **Power BI Desktop** – Data visualization and dashboard development  
- **Power Query** – Data cleaning and ETL (Extract, Transform, Load) process  
- **DAX (Data Analysis Expressions)** – KPI calculations and business logic  
- **Agile / Scrum Methodology** – Sprint-based team collaboration
